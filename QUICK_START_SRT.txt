🐛 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━

Problem: "ffmpeg not found"
Solution: Install FFmpeg and add to PATH
  https://ffmpeg.org/download.html

Problem: "No matching media file"
Solution: Ensure SRT and media files have same name
  ✅ video.srt + video.mp4
  ❌ video.srt + video_HD.mp4

Problem: Import is slow
Solution: Large videos take time. Extract audio first:
  ffmpeg -i video.mp4 -vn -ar 22050 -ac 1 audio.wav

Problem: Too many invalid segments
Solution: Check SRT timing accuracy and audio quality
